# ✅ To-Do List App

A modern and responsive **To-Do List Application** built using **HTML, CSS, and JavaScript**. This project helps users organize daily tasks efficiently by allowing them to add, complete, and delete tasks through a simple and user-friendly interface.

---

## 🚀 Features

* ➕ Add New Tasks
* ✅ Mark Tasks as Completed
* 🗑️ Delete Tasks
* ✏️ Edit Existing Tasks
* 💾 Save Tasks Using Local Storage
* 📱 Fully Responsive Design
* 🎨 Clean and Modern User Interface
* ⚡ Fast and Interactive User Experience

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
